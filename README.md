# Kristina Nazarjanová — Portfolio

Personal portfolio website of Kristina Nazarjanová, UX Designer based in Brno, Czech Republic.


## Stack

- React 19 + TypeScript
- Vite + Tailwind CSS
- Cloudflare Pages

## Run locally

```bash
npm install
npm run dev
```

## Deploy

Deployed via Cloudflare Pages. Every push to `main` triggers an automatic build.
